PROMPT — ablute_ Health Mode

Design and build the Health Mode section of the ablute_ website.

Health Mode represents the scientific, biomedical and preventive health dimension of the ablute_ platform.

The objective is to communicate that ablute_ is not only a sanitation device but also a biosensing platform capable of generating health insights from daily routines.

The interface must feel credible, medical-grade, and scientifically grounded, while still maintaining a modern technology aesthetic.

IMPORTANT:

The agent must analyze the attached design reference images and follow their visual style, layout logic and component hierarchy.

The images illustrate:

• medical dashboard visual style
• biomarker monitoring interface
• modern health-tech UI patterns
• information cards and clean layouts

The Health Mode design must resemble these examples.

VISUAL STYLE

Health Mode must differ from the Bathroom/Product mode.

Use a medical interface design language.

Visual characteristics:

light backgrounds
soft gradients
clinical blues and teals
clean white cards
subtle shadows
clear typography hierarchy

UI should resemble:

modern digital health platforms
medical monitoring dashboards
preventive health applications.

Avoid dark heavy product design used in Bathroom Mode.

Health Mode should feel clinical, structured and trustworthy.

HEALTH MODE MAIN PAGE

The main page introduces the concept of transforming the bathroom into a health interface.

Headline example concept:

Transforming bathrooms into health interfaces

Supporting text should explain that the ablute_ platform integrates sensors and data analysis to enable continuous, passive health monitoring.

HERO VISUAL

The hero visual should resemble the reference images.

Use a mobile health dashboard interface displaying:

health score
biomarker indicators
trend bars
health insights.

The visual should resemble a digital health app interface.
